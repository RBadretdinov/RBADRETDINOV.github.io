Custom Macro - Auto Clicker

A Python GUI application developed to assist with automating repetitive mouse clicking tasks, originally created for a friend of mine to help maintain user presence during idle periods in videogames and other applications. Features an intuitive interface with Start/Stop controls and status indicators, allowing users to configure automatic left mouse clicks at random intervals. The click interval can be customized by the user or defaults to a random range between 7-15 seconds, providing a helpful solution for users who need to maintain activity without manual intervention.

PNGs to PDF Converter

This is a Python command-line tool designed to convert multiple image files (PNG, JPG, JPEG) into a single multi-page PDF document, or merge multiple existing PDF files into one. Useful for creating PDFs from photo collections, streamlining scanning workflows, or combining document files. Supports directory scanning and automatic image format conversion to RGB for compatibility. While its core function is simple, the program was extremely useful because I had originally created it to help submit my high school assignments. Seeing as other PDF converters weren't free, I decided to make my own.
