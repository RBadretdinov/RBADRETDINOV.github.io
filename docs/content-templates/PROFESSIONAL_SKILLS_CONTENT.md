## Certification Template

For each certification, you'll need:
- **Certification Name:** Full name of the certification
- **Date Completed:** When you earned the certification (optional)
- **Description:** 2-3 sentences describing your skills and capabilities related to this certification
- **Image:** Filename of the certification badge/image (stored in `assets/images/certifications/`)

**Format Guidelines:**
- Start descriptions with "I can..." or similar active language
- Be specific about skills and capabilities
- Keep descriptions concise but informative (2-4 sentences)
- Mention what the certification enables you to do
- Distinguish clearly between similar certifications (e.g., two communication certifications)

## Adding New Certifications

To add a new professional certification:
1. Copy a certification template above
2. Fill in the name, date, image filename, and description
3. Ensure the image file is in `assets/images/certifications/`
4. If you have multiple certifications of the same type, make sure descriptions clearly distinguish them

## Notes

- Certification images should be square (600x600px recommended)
- Descriptions should be professional and highlight specific skills
- If you have similar certifications, explain how they differ
- Keep descriptions varied and engaging
- Dates are optional but can add credibility
